Informe de Resultados

Tras realizar un muestreo de 20 mediciones sobre el punto de acceso, se observa que la red presenta una exactitud aceptable pero una precisión baja.

Aunque el valor promedio de las mediciones se mantiene relativamente cerca del estándar óptimo de 100 ms, existe una alta dispersión en los resultados individuales. Contamos con fluctuaciones que van desde un mínimo de 85 ms hasta picos de 120 ms, lo que representa un rango de variación de 35 ms.

Si bien la conexión cumple con el promedio prometido, es inestable: el usuario experimentará picos de latencia o inconsistencias durante el uso continuo. Para que la red sea de alta calidad, se debe reducir el margen de error entre cada medición para lograr una mayor estabilidad en el servicio.
