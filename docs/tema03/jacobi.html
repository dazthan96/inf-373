Metodo Jacobi

El Sistema de Ecuaciones Original

Tomaremos un sistema de 3x3 donde la diagonal principal es estrictamente dominante para asegurar que el método funcione:

(Ecuación 1)   10x₁ - 1x₂ + 2x₃ = 6
(Ecuación 2)  -1x₁ + 11x₂ - 1x₃ = 25
(Ecuación 3)   2x₁ - 1x₂ + 10x₃ = -11

Despeje de las Variables

Despejamos x₁ de la primera ecuación, x₂ de la segunda y x₃ de la tercera. Las fórmulas resultantes son las que usaremos en cada iteración:

x₁ = [ 6 - (-1x₂) - (2x₃) ] / 10

x₂ = [ 25 - (-1x₁) - (-1x₃) ] / 11

x₃ = [ -11 - (2x₁) - (-1x₂) ] / 10

Proceso de Iteración

Para comenzar el método, necesitamos proponer valores iniciales. Usaremos el vector nulo estándar: x₁ = 0, x₂ = 0, x₃ = 0.

Regla de Jacobi: Durante una iteración completa, las operaciones del lado derecho de la igualdad SIEMPRE utilizan los valores de la iteración anterior. No se mezclan valores nuevos.
Iteración 1 (k = 1)

// Sustituimos x₁ = 0, x₂ = 0, x₃ = 0 en los despejes:

x₁⁽¹⁾ = [ 6 - (-1 × 0) - (2 × 0) ] / 10
x₁⁽¹⁾ = [ 6 - 0 - 0 ] / 10 = 6 / 10 = 0.60000
x₂⁽¹⁾ = [ 25 - (-1 × 0) - (-1 × 0) ] / 11
x₂⁽¹⁾ = [ 25 - 0 - 0 ] / 11 = 25 / 11 = 2.27273
x₃⁽¹⁾ = [ -11 - (2 × 0) - (-1 × 0) ] / 10
x₃⁽¹⁾ = [ -11 - 0 - 0 ] / 10 = -11 / 10 = -1.10000
Iteración 2 (k = 2)

// Ahora sustituimos los resultados anteriores: x₁ = 0.60000, x₂ = 2.27273, x₃ = -1.10000

x₁⁽²⁾ = [ 6 - (-1 × 2.27273) - (2 × -1.10000) ] / 10
x₁⁽²⁾ = [ 6 + 2.27273 + 2.20000 ] / 10
x₁⁽²⁾ = [ 10.47273 ] / 10 = 1.04727
x₂⁽²⁾ = [ 25 - (-1 × 0.60000) - (-1 × -1.10000) ] / 11
x₂⁽²⁾ = [ 25 + 0.60000 - 1.10000 ] / 11
x₂⁽²⁾ = [ 24.50000 ] / 11 = 2.22727
x₃⁽²⁾ = [ -11 - (2 × 0.60000) - (-1 × 2.27273) ] / 10
x₃⁽²⁾ = [ -11 - 1.20000 + 2.27273 ] / 10
x₃⁽²⁾ = [ -9.92727 ] / 10 = -0.99273

Metodo Seidel

El Sistema de Ecuaciones Original

Utilizaremos exactamente el mismo sistema de 3x3 del ejercicio anterior para que puedas comparar ambos comportamientos:

(Ecuación 1)   10x₁ - 1x₂ + 2x₃ = 6
(Ecuación 2)  -1x₁ + 11x₂ - 1x₃ = 25
(Ecuación 3)   2x₁ - 1x₂ + 10x₃ = -11

Despeje de las Variables

Los despejes matemáticos son idénticos a los de Jacobi. Lo que cambiará por completo es el orden en el que introduciremos los datos:

x₁ = [ 6 - (-1x₂) - (2x₃) ] / 10

x₂ = [ 25 - (-1x₁) - (-1x₃) ] / 11

x₃ = [ -11 - (2x₁) - (-1x₂) ] / 10

Proceso de Iteración

Asumimos los mismos valores iniciales de partida: x₁ = 0, x₂ = 0, x₃ = 0.

Regla de Gauss-Seidel: A diferencia de Jacobi, en cuanto calculamos un nuevo valor para una variable, lo usamos DE INMEDIATO para calcular las siguientes variables dentro de esa misma iteración.
Iteración 1 (k = 1)

// Calculamos x₁ usando los valores iniciales (x₂ = 0, x₃ = 0):

x₁⁽¹⁾ = [ 6 - (-1 × 0) - (2 × 0) ] / 10
x₁⁽¹⁾ = [ 6 - 0 - 0 ] / 10 = 6 / 10 = 0.60000

// Calculamos x₂ usando el NUEVO x₁ (0.60000) y el viejo x₃ (0):

x₂⁽¹⁾ = [ 25 - (-1 × 0.60000) - (-1 × 0) ] / 11
x₂⁽¹⁾ = [ 25 + 0.60000 - 0 ] / 11 = 25.6 / 11 = 2.32727

// Calculamos x₃ usando el NUEVO x₁ (0.60000) y el NUEVO x₂ (2.32727):

x₃⁽¹⁾ = [ -11 - (2 × 0.60000) - (-1 × 2.32727) ] / 10
x₃⁽¹⁾ = [ -11 - 1.20000 + 2.32727 ] / 10
x₃⁽¹⁾ = [ -9.87273 ] / 10 = -0.98727
Iteración 2 (k = 2)

// Calculamos x₁ usando los últimos valores disponibles (x₂ = 2.32727, x₃ = -0.98727):

x₁⁽²⁾ = [ 6 - (-1 × 2.32727) - (2 × -0.98727) ] / 10
x₁⁽²⁾ = [ 6 + 2.32727 + 1.97454 ] / 10
x₁⁽²⁾ = [ 10.30181 ] / 10 = 1.03018

// Calculamos x₂ usando el NUEVO x₁ (1.03018) y el viejo x₃ (-0.98727):

x₂⁽²⁾ = [ 25 - (-1 × 1.03018) - (-1 × -0.98727) ] / 11
x₂⁽²⁾ = [ 25 + 1.03018 - 0.98727 ] / 11
x₂⁽²⁾ = [ 25.04291 ] / 11 = 2.27663

// Calculamos x₃ usando el NUEVO x₁ (1.03018) y el NUEVO x₂ (2.27663):

x₃⁽²⁾ = [ -11 - (2 × 1.03018) - (-1 × 2.27663) ] / 10
x₃⁽²⁾ = [ -11 - 2.06036 + 2.27663 ] / 10
x₃⁽²⁾ = [ -10.78373 ] / 10 = -1.07837
