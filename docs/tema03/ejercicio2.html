Ejercicio 2 Volver al Inicio

Desafio 1

Contexto Una empresa factura servicios de Estrategia (x), Diseño (y) y Soporte (z). Para equilibrar sus costos y asegurar estabilidad numérica, las metas son:

  • Predominio de Estrategia: El cuádruple de la Estrategia más el Diseño y el Soporte suman $22,000.
  • Predominio de Diseño: La Estrategia más el quíntuple del Diseño más el doble del Soporte suman $33,000.
  • Predominio de Soporte: La Estrategia más el Diseño más el triple del Soporte suman $19,000.

Metodo Jacobi

El Sistema de Ecuaciones Original

Tomaremos un sistema de 3x3 donde la diagonal principal es estrictamente dominante para asegurar que el método funcione:

(Ecuación 1)   4x₁ + 1x₂ + 1x₃ = 22000
(Ecuación 2)   1x₁ + 5x₂ + 2x₃ = 33000
(Ecuación 3)   1x₁ + 1x₂ + 3x₃ = 19000

Despeje de las Variables

Despejamos x₁ de la primera ecuación, x₂ de la segunda y x₃ de la tercera. Las fórmulas resultantes son las que usaremos en cada iteración:

x₁ = [ 22000 -1x₂ - 1x₃ ] / 4

x₂ = [ 33000 -1x₁ -2x₃ ] / 5

x₃ = [ 19000 - 1x₁ -1x₂ ] / 3

Proceso de Iteración

Para comenzar el método, necesitamos proponer valores iniciales. Usaremos el vector nulo estándar: x₁ = 0, x₂ = 0, x₃ = 0.

Regla de Jacobi: Durante una iteración completa, las operaciones del lado derecho de la igualdad SIEMPRE utilizan los valores de la iteración anterior. No se mezclan valores nuevos.
Iteración 1 (k = 1)

Sustituimos x₁ = 0, x₂ = 0, x₃ = 0 en los despejes:

x₁ = 5500
x₂ = 6600
x₃ = 6333.33
Iteración 2 (k = 2)

Sustituimos x₁ = 5500, x₂ = 6600, x₃ = 6333.33 en los despejes:

x₁ = 2266.67
x₂ = 2966.67
x₃ = 2300.00
Iteración 2 (k = 2)

En un archivo excel, hasta donde se decida, nosotros iteramos 10 veces teniendo los siguientes resultados:

x₁ = 3449.80
x₂ = 4404.36
x₃ = 3679.95

Metodo Seidel

El Sistema de Ecuaciones Original

Tomaremos un sistema de 3x3 donde la diagonal principal es estrictamente dominante para asegurar que el método funcione:

(Ecuación 1)   4x₁ + 1x₂ + 1x₃ = 22000
(Ecuación 2)   1x₁ + 5x₂ + 2x₃ = 33000
(Ecuación 3)   1x₁ + 1x₂ + 3x₃ = 19000

Despeje de las Variables

Despejamos x₁ de la primera ecuación, x₂ de la segunda y x₃ de la tercera. Las fórmulas resultantes son las que usaremos en cada iteración:

x₁ = [ 22000 -1x₂ - 1x₃ ] / 4

x₂ = [ 33000 -1x₁ -2x₃ ] / 5

x₃ = [ 19000 - 1x₁ -1x₂ ] / 3

Proceso de Iteración

Asumimos los mismos valores iniciales de partida: x₁ = 0, x₂ = 0, x₃ = 0.

Regla de Gauss-Seidel: A diferencia de Jacobi, en cuanto calculamos un nuevo valor para una variable, lo usamos DE INMEDIATO para calcular las siguientes variables dentro de esa misma iteración.
Iteración 1 (k = 1)

// Calculamos x₁ usando los valores iniciales (x₂ = 0, x₃ = 0):

x₁ = 5500

// Calculamos x₂ usando el NUEVO x₁ (5500) y el viejo x₃ (0):

x₂ = 5500

// Calculamos x₃ usando el NUEVO x₁ (5500) y el NUEVO x₂ (5500):

x₃ = 2666.67
Iteración 2 (k = 2)

// Calculamos x₁ usando los últimos valores disponibles (x₂ = 5500, x₃ = 2666.67):

x₁ = 3458.33

// Calculamos x₂ usando el NUEVO x₁ (3458.33) y el viejo x₃ (2666.67):

x₂ = 4841.67

// Calculamos x₃ usando el NUEVO x₁ (3458.33) y el NUEVO x₂ (4841.67):

x₃ = 3566.67
Despues de 10 iteraciones
x₁ = 3468.08
x₂ = 4425.53
x₃ = 3702.13

Intepretacion

Considerando los montos obtenidos en el contexto de la consultora, la interpretación es la siguiente:

Para cumplir con las metas de facturación y los compromisos contractuales establecidos, la empresa debe distribuir su operación de la siguiente manera:

  • Estrategia: Debe generar un ingreso de $3,468.08. Este departamento actúa como el motor de apertura, teniendo una carga operativa moderada pero una meta de facturación específica que cumplir.
  • Diseño: Es el área con mayor actividad económica, requiriendo una facturación de $4,425.53. Esto indica que el Diseño es el servicio más demandado o el que mayor volumen de trabajo aporta al flujo de caja total de la consultora.
  • Soporte: Debe aportar $3,702.13. Este valor representa el mantenimiento y la atención post-venta necesarios para equilibrar los costos operativos fijos de la estructura.

Para que la consultora alcance el equilibrio financiero proyectado de los $74,000 totales (suma de las tres metas), el Diseño debe ser la prioridad comercial, seguido de cerca por el Soporte, dejando a la Estrategia como un componente de alto valor pero de menor volumen transaccional.
