Propagación

El propósito de esta sección consiste en estudiar cómo los errores en los números pueden propagarse a través de las funciones matemáticas. Por ejemplo, si se multiplican dos números que tienen errores, nos gustaría estimar el error de este producto.

Para lograr esto, el proceso se resume en:

  1. Se debe tener una función que contenga mas de 2 variables.
  2. Se debe tener los valores para la funcion (para la anotación) como tambien la variación de cada una de sus variables
  3. Derivar la función para cada una de las variables de contenga.
  4. Multiplicar la funcion derivada con respecto a un variable y multiplicarla con la variacion de su respectiva variable.
  5. Sumar cada resultado de la operacion anterior.
